# ty-pre-commit

## Pre-commit hook

Add this hook to your `.pre-commit-config.yaml`:

```yaml
repos:
- repo: https://github.com/narumiruna/ty-pre-commit
  rev: v0.0.7a35a34a33a32a31a30a29a28a27a26a25a24a23a22a21a20a19a18a17a16a15a14a13a12a11a10a9a8a7a6a5a4a3a2a1a35a34a33a32a31a30a29a28a27a26a25a24a23a22a21a20a19a18a17a16a15a14a13a12a11a10a9a8a7a6a5a4a3a2a1
  hooks:
    - id: ty-check
```
